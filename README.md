# We got jobs - Your ultimate freelance market

Full-stack freelance marketplace built with **Next.js 15 + NestJS 11 + Prisma 7 + Turborepo + better-auth**

- Frontend → http://localhost:3000  
- Backend  → http://localhost:3001  

## Tech Stack

| Layer            | Technology                                      |
|------------------|-------------------------------------------------|
| Monorepo         | Turborepo + pnpm workspaces                     |
| Frontend         | Next.js 15 (App Router) + Tailwind + TypeScript|
| Backend          | NestJS 11 + TypeScript                          |
| Database         | PostgreSQL + Prisma 7 (shared client)           |
| Auth             | better-auth    |
| Real-time (soon) | Socket.io + Redis                               |
| Queue (soon)     | BullMQ + Redis                                  |
| Payments (soon)  | Stripe Connect                                  |
| UI               | shadcn/ui + Radix UI                            |
